# 依次执行多个异步函数

## 构建队列

```javascript
function excute (list) {
    if (!list || list.length < 1) {
        return;
    }
    let lastOne = Promise.resolve();
    for (let i=0, len=list.length; i<len; i++) {
        lastOne = lastOne.then(list[i]);
    }
}
```

## 使用async/await

```javascript
async function excute (list) {
    if (!list || list.length < 1) {
        return;
    }
    for (let i=0, len=list.length; i<len; i++) {
        await list[i]();
    }
}
```
